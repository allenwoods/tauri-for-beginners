# 边框

在 CSS 中，可以通过设置 border-width 属性来指定边框宽度。

#### 示例

```css
header {
    border-width: 5px;
}
```

可以通过指定 border-color 属性来设置边框颜色，如下所示。

#### 示例

```css
header {
    border-color: red;
}
```

border-style 属性控制边框是点状、虚线、实线等。可以按如下所示的方式完成。

#### 示例

```css
header {
    border-style: dotted;
}
```

可以使用简写形式指定边框属性，如下所示。

#### 示例

```css
header {
    border: 1px solid black;
}
```

这将创建一个 1 像素厚、实心黑色的边框。
